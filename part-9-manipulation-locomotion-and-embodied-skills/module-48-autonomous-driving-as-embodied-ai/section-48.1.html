Theory

The minimal formal object is a partially observed control loop. At time $t$, the agent receives $o_t$, estimates $\hat s_t$, chooses action $a_t$, and receives a new observation $o_{t+1}$. In Driving as perception, prediction, planning, control, the important state variables are sensors, map, actor state, route intent, and vehicle dynamics, while the action interface is to predict, plan, and control as one embodied loop.

The method is useful only if it improves a construct-matched measurement: route completion and safety margin. That metric must be co-computed for the baseline and the library shortcut on the same scenario panel, with the same seed policy and the same termination rules.

Mechanism

Driving autonomy is a timed loop: sensor fusion produces tracked actors, prediction estimates futures, planning selects a maneuver, control tracks it, and monitors decide when to disengage.
